== Description ==
Utopia theme is a clean responsive wordpress theme under GPLv2 Licence, built with bootstrap and contains many features to enhance your wordpress blog into a real website.A complete customizer options and a theme options page that let you personnalize front and back end.


== Installation ==


1. Upload `utopia`  theme to the `/wp-content/theme/` directory
2. Activate the theme through the 'theme' menu in WordPress


== Frequently Asked Questions ==

1-How can I change Front End display ?
All front end possible changes are made via wordpress customizer which alow you to change your logo, site tiltle, colours, sibebar layout and many more...

2-How can I change Back end Features?
Utopia theme options page , that you can find under appearance menu tab, lets you change many things on the background like removing default widgets, changing admin screen logo and links, and many more
